# Outputs Reference

Complete catalog of all generated files with schemas and descriptions.

## data/processed/

Cleaned datasets ready for modeling. Generated by `src/data/preprocessing.py`.

| File | Rows | Description |
|------|-----:|-------------|
| `manual_clean.csv` | 5,949 | Deduplicated manual dataset |
| `synthetic_clean.csv` | 7,991 | Deduplicated synthetic dataset |
| `combined_clean.csv` | 13,940 | Concatenation of both |

**Schema:**

| Column | Type | Description |
|--------|------|-------------|
| `dataset_source` | string | "manual" or "synthetic" |
| `label` | string | "ham", "spam", or "smishing" |
| `text` | string | Original SMS text |
| `text_clean` | string | Whitespace-normalized text |
| `has_url` | int | 1 if contains URL, 0 otherwise |
| `has_email` | int | 1 if contains email, 0 otherwise |
| `has_phone` | int | 1 if contains phone number, 0 otherwise |

## outputs/metrics/

### all_model_results.csv

270 rows. One row per seed/model/experiment combination.

| Column | Type | Description |
|--------|------|-------------|
| `seed` | int | Random seed (1–30) |
| `model_name` | string | MultinomialNB, LogisticRegression, or LinearSVC |
| `experiment_id` | string | manual_only, synthetic_only, or combined |
| `duplicate_mode` | string | Always "drop_exact_duplicates" for primary analysis |
| `n_train` | int | Number of training rows |
| `n_test` | int | Number of test rows |
| `accuracy` | float | Overall accuracy |
| `macro_precision` | float | Macro-averaged precision |
| `macro_recall` | float | Macro-averaged recall |
| `macro_f1` | float | Macro-averaged F1 |
| `ham_precision` | float | Ham class precision |
| `ham_recall` | float | Ham class recall |
| `ham_f1` | float | Ham class F1 |
| `spam_precision` | float | Spam class precision |
| `spam_recall` | float | Spam class recall |
| `spam_f1` | float | Spam class F1 |
| `smishing_precision` | float | Smishing class precision |
| `smishing_recall` | float | Smishing class recall |
| `smishing_f1` | float | **Primary metric** — smishing class F1 |

### confusion_matrices.csv

Long-format confusion matrices. 270 runs x 9 cells = 2,430 rows.

| Column | Type | Description |
|--------|------|-------------|
| `seed` | int | Random seed |
| `model_name` | string | Classifier name |
| `experiment_id` | string | Training strategy |
| `true_label` | string | Actual class |
| `predicted_label` | string | Predicted class |
| `count` | int | Number of instances |

### classification_reports.jsonl

270 lines, one JSON object per line. Each contains:

```json
{
  "seed": 1,
  "model_name": "MultinomialNB",
  "experiment_id": "manual_only",
  "report": { ... sklearn classification_report dict ... }
}
```

## outputs/tables/

### Dataset Audit Tables

| File | Description |
|------|-------------|
| `dataset_audit.csv` | Per-dataset summary: row count, class counts, duplicate stats, avg message length, vocab size |
| `cross_dataset_overlap.csv` | Every synthetic text that appears verbatim in the manual dataset |
| `top_duplicate_messages.csv` | Top 20 most duplicated messages across both datasets |
| `leakage_audit.csv` | Overlap counts and percentages between datasets |

### Performance Summary Tables

| File | Description |
|------|-------------|
| `model_performance_summary.csv` | Metrics aggregated by model_name (mean, std, median, min, max, IQR) |
| `experiment_performance_summary.csv` | Metrics aggregated by experiment_id |

### Statistical Analysis Tables

| File | Description |
|------|-------------|
| `statistical_tests.csv` | Paired test results: comparison, metric, model_name, test_type, n_pairs, mean_difference, median_difference, statistic, p_value, normality_p, p_value_adjusted, significant_at_0_05 |
| `effect_sizes.csv` | Effect sizes: comparison, metric, model_name, cohens_d, cohens_d_magnitude, wilcoxon_r |
